# 引注自查表

> 投稿前逐项核对。对照《法学引注手册》（2020年版）和本 skill 的 `references/legal-citation.md`。

---

## 一、法律文件引注

### 法律
- [ ] 法律名称首次出现时使用全称（`《中华人民共和国XX法》`）？
- [ ] 引用《民法典》《刑法》等常用法律时，是否标注了施行/修正年份？
- [ ] 如有引用已废止的法律（如原《合同法》），是否标注了废止信息？
- [ ] 法条编号准确（条/款/项/目）？

### 行政法规
- [ ] 是否标注了国务院令号？
- [ ] 是否标注了施行/修正年份？

### 司法解释
- [ ] 是否标注了文号（法释〔20XX〕X号）？
- [ ] 该司法解释是否现行有效（未被废止或替代）？
- [ ] 是否区分了"司法解释"与"部门规范性文件"？

### 部门规章
- [ ] 是否标注了发布机关和令号？

### 地方性法规
- [ ] 是否标注了发布机关（省/市人大及其常委会）？

### 国际条约
- [ ] 是否注明了通过/生效日期？
- [ ] 是否提供了条约全称和通用简称（如CISG）？

---

## 二、判例引注

### 指导性案例
- [ ] 是否注明了编号（"指导案例第XX号"）？
- [ ] 是否注明了案件名称？
- [ ] 是否注明了发布日期？

### 公报案例
- [ ] 是否注明了《最高人民法院公报》年份和期号？

### 普通案例
- [ ] 是否标注了完整案号（含年份、法院简称、审级、类型、编号）？
- [ ] 审理法院是否使用了全称（非简称）？
- [ ] 是否标注了裁判日期（年/月/日）？
- [ ] 是否区分了判决书和裁定书？

---

## 三、文献引注

### 期刊论文
- [ ] 作者姓名是否完整？
- [ ] 论文标题是否准确（含副标题）？
- [ ] 期刊名称是否使用全称（非缩写）？
- [ ] 是否标注了年份、期号、引用页码？
- [ ] 如为译文，是否注明了译者？

### 专著
- [ ] 作者/编者信息是否完整？
- [ ] 书名是否准确（含副标题）？
- [ ] 是否标注了出版社和出版年份？
- [ ] 是否标注了引用页码？
- [ ] 如为译著，是否注明了译者？

### 文集/编著
- [ ] 论文标题是否准确？
- [ ] 主编/编者信息是否完整？
- [ ] 文集名称是否准确？
- [ ] 是否标注了出版社、出版年份、引用页码？

### 学位论文
- [ ] 是否标注了学校名称和学位层级（硕士/博士）？
- [ ] 是否标注了年份？
- [ ] 如有DOI，是否标注？

### 网络资料
- [ ] 是否标注了作者/发布机构？
- [ ] 是否标注了标题和网站名称？
- [ ] 是否标注了发布日期？
- [ ] **是否标注了访问日期？**（最容易被遗漏的项）
- [ ] URL 是否有效（未失效）？

---

## 四、引注格式一致性

- [ ] 全篇引注风格统一（遵循《法学引注手册》，非混用 GB/T 7714）？
- [ ] 脚注编号连续，无重复/跳跃？
- [ ] 同一文献多次引用时，"见前注"格式是否正确使用？
- [ ] 紧接引用同一文献时，"同上注"格式是否正确使用？
- [ ] 中文标点（、。「」『』）与英文标点（, . "") 未混用？
- [ ] 外文文献的标点/斜体/大写等格式是否一致？
- [ ] 参考文献列表（如需要）是否与脚注对应？

## 六、目标期刊特殊规则

> 投稿前对照目标期刊的"投稿须知"或"注释体例"。

- [ ] 已下载目标期刊近期的"投稿须知"/"注释体例"？
- [ ] 已核对目标期刊的特殊要求（如是否接受"见前注"格式、参考文献是否需要单列等）？
- [ ] 已参照目标期刊最近一期的引注格式做最后核对？
