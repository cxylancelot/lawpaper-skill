# 法学论文大纲模板

> 按 6 种论文类型各提供一个可填空的大纲骨架。"[ ]" 中的内容替换为你的具体内容。

---

## 模板 A：法教义学论文

```
标题：[核心概念/法条]：[论证方向]

摘要：本文讨论 [核心问题]。通过对 [核心法条] 的文义、体系和目的解释，本文认为 [核心论点]。（150-200字）

引言
- 第1段：[具体争议/分歧现象] → 引出核心问题
- 第2段：已有研究的不足 + 本文的切入点
- 第3段：本文试图论证 [核心论点]。结构如下：……

一、[核心法条/制度]的规范结构
  1.1 法条文本与体系位置
  1.2 与该条相关的配套规范
  1.3 规范结构中蕴含的解释问题

二、现有解释方案的梳理与评析
  2.1 方案A：[简述+支持理由+局限]
  2.2 方案B：[简述+支持理由+局限]
  2.3 方案C（如有）：[简述]
  2.4 小结：争议的焦点在于 [核心分歧]

三、本文的解释方案
  3.1 文义解释：[该概念的法条文义分析]
  3.2 体系解释：[该概念在其他法条、章节中的用法及体系功能]
  3.3 目的解释：[立法目的的探究] + [为什么本方案最符合立法目的]
  3.4 对反对意见的回应

四、结论
  4.1 [核心法条]中的 [核心概念] 应理解为 [本文结论]
  4.2 该结论对司法实践的指导意义
  4.3 本文的局限（如有）
```

## 模板 B：案例分析

```
标题：[案件名称/类型]中的 [核心法律问题]：[论证方向]

摘要：本文以 [案件名称/类型] 为分析对象，讨论 [核心法律问题]。本文认为 [核心论点]。（150-200字）

引言
- 第1段：[案件简要介绍] + 该案触及的 [核心法律问题]
- 第2段：该问题在已有研究中的讨论状态
- 第3段：本文试图通过对该案的深入分析，论证 [核心论点]

一、案情与争点
  1.1 基本案情（精简到只保留与争点相关的事实）
  1.2 裁判过程与结果
  1.3 核心争点提炼：[1-3个法律问题]

二、裁判逻辑分析
  2.1 法院对争点[1]的裁判逻辑
  2.2 法院对争点[2]的裁判逻辑
  2.3 裁判逻辑的综合评价

三、法理评析
  3.1 [评价法院裁判的正确/可商榷/错误之处]
  3.2 [与其他类似案件的裁判逻辑对比]（如有）
  3.3 [本案裁判逻辑对 [同类案件] 的指导性/不可指导性]

四、裁判规则的提炼及其规范意义
  4.1 本案可提炼的裁判规则：[具体表述]
  4.2 该规则对类似情形的适用条件
  4.3 该规则的法理基础与限度

五、结论
```

## 模板 C：立法评述

```
标题：[法律名称]的 [修改/新设]：[制度评析方向]

摘要：[新法/修法]回应了 [现实问题]，核心制度创新在于 [X]。本文认为 [评价+建议]。（150-200字）

引言
- 第1段：[立法背景] + 为何需要关注 [这部法律/这次修改]
- 第2段：已有评述的侧重点和不足
- 第3段：本文聚焦 [1-2个核心制度]，依次评析并给出完善建议

一、立法背景与制度变化
  1.1 立法背景与动因
  1.2 新旧法核心制度对比（表格形式）
  1.3 本次立法的核心制度创新

二、[核心制度A]的评析
  2.1 制度设计逻辑
  2.2 进步之处
  2.3 可商榷之处（具体问题+法理分析）

三、[核心制度B]的评析
  （同上结构）

四、遗留问题与完善方向
  4.1 修法后仍未解决的问题
  4.2 完善建议（具体到规则层面）
  4.3 后续配套制度建设建议

五、结论
```

## 模板 D：比较法论文

```
标题：[域外制度]的 [核心要素] 及其对中国 [相关领域] 的启示

摘要：[域外法域]的 [制度] 在 [核心方面] 为中国 [相关领域] 提供了 [分析参照/借鉴资源]。本文认为，在 [条件]下，中国可以借鉴 [具体要素]；但 [限缩条件]。（150-200字）

引言
- 第1段：中国面临的法律问题 + 该问题的域外参照价值
- 第2段：已有比较法研究的侧重点和不足
- 第3段：为什么选择 [该法域] 作为比较对象 + 本文的论证路径

一、[域外法域]的 [制度]：[中国问题导向的选择性分析]
  1.1 该制度的基本框架（聚焦与中国问题相关的部分）
  1.2 该制度的核心运行逻辑
  1.3 该制度运行的条件与环境

二、中国相关制度的功能对应与差异
  2.1 中国面临的问题与 [域外法域] 的制度功能对应
  2.2 两国制度条件的异同
  2.3 制度差异的深层原因

三、借鉴与限缩
  3.1 可借鉴的要素及其理由
  3.2 不可借鉴/不适合直接移植的要素及其理由
  3.3 借鉴后的中国方案（兼顾本土制度条件的调整）

四、结论
```

## 模板 E：实证研究（裁判文书大数据）

```
标题：[研究现象]：[核心发现]

摘要：基于对 [N] 份 [案由] 判决书的分析，本文发现 [核心发现1]、[核心发现2]。这些发现表明 [理论/实践意义]。（150-200字）

引言
- 第1段：[研究现象的普遍印象] vs [实际可能并非如此] 的反差
- 第2段：已有研究对此问题的讨论（主要是规范分析）及其局限
- 第3段：本文的研究问题+假设+方法概述

一、问题的提出与研究假设
  1.1 制度背景与研究问题
  1.2 既有研究的贡献与不足
  1.3 研究假设

二、数据与方法
  2.1 数据来源与检索条件（案由/时间/法院层级/检索平台）
  2.2 筛选标准与最终样本量
  2.3 分析方法（编码方案/变量设定/统计方法）
  2.4 方法的局限

三、发现
  3.1 描述性统计（整体分布、趋势等）
  3.2 核心发现1：[数据呈现+解释]
  3.3 核心发现2：[数据呈现+解释]
  3.4 其他值得注意的发现

四、讨论
  4.1 发现与已有研究的对话（印证/挑战/修正了什么？）
  4.2 发现对制度理解和实践的意义
  4.3 研究局限对结论的影响

五、结论
```

## 模板 F：法哲学论文

```
标题：[核心概念/命题]：[论证方向]

摘要：[核心问题] 的根源在于 [概念混淆/理论盲区]。本文通过 [概念分析/思想实验]，论证 [核心论点]。（150-200字）

引言
- 第1段：[具体的思想困惑/理论争议] → 引出核心概念/命题
- 第2段：已有理论对该问题的解释及局限
- 第3段：本文的论证路径

一、问题的澄清
  1.1 [核心概念] 的日常理解与理论理解之间的张力
  1.2 已有定义方案的梳理与评析
  1.3 本文的切入点：重新界定 [核心问题] 的性质

二、[概念分析/思想实验/论证]
  （按具体论证逻辑组织）
  2.1 [第一步分析]
  2.2 [第二步分析]
  2.3 [对反对意见的回应]

三、理论意涵与实践意义
  3.1 [核心论证结果] 对 [更大理论命题] 的意义
  3.2 如有实践关联，说明对 [具体法律问题] 的启发

四、结论
```
