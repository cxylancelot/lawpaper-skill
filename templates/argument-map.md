## 核心论点 (Claim)

```

_________________________________________________________________
```

## 推理保证 (Warrant)

> 从每类论据到核心论点的推理桥梁是什么？

```
从规范论据 → 核心论点的推理链：
"规定X+Y+Z，意味着……，因此……"

从判例论据 → 核心论点的推理链：
"这些判决一致/不一致地采用了A标准，说明……，因此……"

从学理论据 → 核心论点的推理链：
"已有研究在……方面达成了共识/产生了分歧，说明……，因此……"
```

## 限定 (Qualifier)

> 你的论点的适用范围和成立条件。

```
我的论点在以下条件下成立：
1. ___________________________________________________
2. ___________________________________________________
3. ___________________________________________________

我的论点在以下情形下不成立（或不完全成立）：
1. ___________________________________________________
2. ___________________________________________________

这些限定是否合理？（如果限定太多——你的论点可能太弱；限定太少——可能被反例推翻）
```

## 论证链检测

> 检查各分论点之间的逻辑关系。

```
分论点1 → 分论点2 → 分论点3 → 核心论点

把以上连接起来读一遍：
"因为 [分论点1]，所以 [分论点2]；在 [分论点2] 的基础上，[分论点3]；因此 [核心论点]。"

这是递进的论证链吗？（是 / 否 → 否的话是"并列"了吗？需要重新设计论证结构）
```
