# 法学论文类型决策树

> 从你的研究起点出发，逐步分叉到最适合的论文类型。回答每个节点的问题，跟踪你的分支。

---

## 决策树

```
你的研究起点是什么？
│
├─ 一个法条的适用争议
│   └─→ 法教义学论文
│       ├─ 如果主要用解释方法（文义/体系/目的/历史）→ 纯法教义学
│       └─ 如果结合大量判例来论证该法条的适用分歧 → 法教义学 + 案例分析
│
├─ 一份具体的判决
│   └─→ 案例分析
│       ├─ 如果深入分析这一份判决的裁判逻辑 → 单一案例评析
│       └─ 如果以这份判决为引子，分析同类案件的裁判规律 → 多案例比较分析
│
├─ 一项新颁布/修改的法律
│   └─→ 立法评述
│       ├─ 如果侧重"这部法律改了什么、为什么这样改" → 立法背景+制度分析型
│       └─ 如果侧重"这部法律实施后有什么问题" → 实施评估型（需要时间积累数据）
│
├─ 一个外国制度的兴趣
│   └─→ 比较法论文
│       ├─ 如果需要回答"中国该不该学、怎么学" → 功能比较
│       ├─ 如果需要解释"为什么同样的制度在两个国家长成了不同的样子" → 结构/历史比较
│       └─ 注意！必须能回答"为什么选这个法域"——不能是"因为我学过这门外语"
│
├─ 一个需要数据回答的实务问题
│   └─→ 实证研究
│       ├─ 如果数据来源是已公开的裁判文书 → 裁判文书大数据分析
│       ├─ 如果数据需要自己采集（问卷/访谈） → 一手实证研究
│       └─ 注意！需要确认数据可得性和样本量在毕业/投稿之前能完成
│
└─ 一个抽象的法理问题
    └─→ 法哲学论文
        ├─ 如果辨析基础概念（权利/义务/责任/正义等） → 概念分析
        ├─ 如果检验某一理论命题 → 思想实验/反思均衡
        └─ 注意！法哲学论文在法学类期刊上的发表空间有限，投稿时注意期刊偏好
```

## 每种类型的输出推荐

| 论文类型 | 推荐结构模式 | 核心论证方法 | 典型篇幅（期刊） | 目标期刊类型 |
|---------|------------|------------|---------------|------------|
| 纯法教义学 | 问题→规范梳理→学说→论证→结论 | 文义/体系/目的/历史解释 | 1.5-2.5万字 | 所有法C刊均接受 |
| 法教义学+案例分析 | 同上，论证部分以判例为材料 | 解释方法为主，判例分析辅助 | 1.5-2.5万字 | 实务型/综合型期刊偏好 |
| 单一案例评析 | 案情→争点→裁判逻辑→评析→规范意义 | 裁判逻辑评价 | 1-1.5万字（短） | 《法律适用》《人民司法》等 |
| 多案例比较分析 | 问题→案例筛选→裁判要旨提取→类型化→规范命题 | 类型化分析 | 1.5-2.5万字 | 所有法C刊，需要强论证 |
| 立法评述 | 背景→制度变化→评析→完善 | 制度分析 | 1-2万字 | 综合型期刊，时效性强 |
| 功能比较法 | 问题→域外法→比较→借鉴与限缩 | 功能比较 | 1.5-2.5万字 | 理论型期刊偏好 |
| 裁判文书大数据 | 问题与假设→数据与方法→发现→讨论→局限 | 定量分析 | 1.5-2.5万字 | 接受实证方法的期刊 |
| 一手实证研究 | 同上，数据自采 | 定量/定性分析 | 2-3万字 | 接受实证方法的期刊 |
| 法哲学论文 | 问题→概念分析/思想实验→结论 | 哲学方法 | 1.5-2.5万字 | 理论型期刊，《法学研究》《中国法学》等 |

## 边界不清时的判断辅助

### "我有法条也有判决——做法教义学还是案例分析？"

- 法条是核心研究对象，判决是材料 → **法教义学**
- 判决是核心研究对象，法条是评价标准 → **案例分析**
- 两者同等重要 → **法教义学+案例分析**（但需要清晰的论证主线，不要让读者困惑到底在做什么）

### "我有中国也有外国的材料——做法教义学还是比较法？"

- 核心问题是"中国法应该怎么解释" → **法教义学**（比较法材料作为辅助论证）
- 核心问题是"中国法应该怎么改" → **比较法**（但需要先做法教义学分析——现有法为什么不够）
- 比较法论文的三个"必须有"：必须说明为什么选这个法域、必须分析两国制度条件的异同、必须说明借鉴的限度

### "我想做实证但怕方法不过关——怎么办？"

- 先确认：你的核心问题是规范性的还是经验性的？
  - 如果是规范性的（"这个法条应该怎么解释"）→ 不需要实证，做法教义学
  - 如果是经验性的（"这个法条在实践中怎么被用"）→ 需要实证
- 如果需要进行实证但自己方法不过关：
  - 裁判文书大数据分析是法学实证研究的最低门槛（无需学新工具，Excel即可开始）
  - 不要为了"看起来更科学"硬上自己不会的方法
