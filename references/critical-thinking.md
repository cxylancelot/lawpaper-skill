## 一、什么是批判性思维

批判性思维（Critical Thinking）是指**识别、分析、评估观点和事实，通过令人信服的推理阐明有证据支撑的结论，能够察觉并克服个人在意识层面的个人中心主义和认识缺陷**，在问题的解决、观点的评判、信念的辨别等方面作出合理明智决策的过程中所需要的一系列认知技能和思维素质的总称。

在中国教育语境下，批判性思维的培养长期不足。一个经典对比：犹太父母在孩子放学后问"你今天问了什么问题？有没有问到老师回答不了的？"，中国父母通常问"老师今天讲了什么？你记住了什么？"。前者培养问题意识——批判性思维的起点；后者停留在知识掌握层面。

**批判性思维≠批判**。它不是简单地否定或攻击他人观点，而是在理性框架下：
- 识别问题所在
- 分析已有观点和证据
- 评估论证的有效性
- 得出基于证据的结论

## 二、批判性思维的核心：论证

论证是批判性思维的核心操作。从批判性思维角度理解，论证不仅是逻辑形式，更是在**人际交流**和**问题解决**的背景下展开的：

### 论证的五维理解

| 维度 | 含义 | 法学写作启示 |
|------|------|------------|
| **与人交流** | 论证发生在他者之间，目的是向他人传递观点并争取认同 | 论文写给学术同行看，不是写给零基础读者 |
| **使人相信** | 理由必须是"使别人相信"的客观理由，而非"自己相信"的主观理由 | 区分"给出原因"和"推理证明"——学位论文常见通病 |
| **与知识联系** | 论证的理由来源于专业知识和对世界的正确认知 | 引注是论证的有机部分，不是装饰 |
| **由问题引发** | 论证总是在一个问题被触发后启动 | 没有问题→不需要论证；论文必须有真正的问题 |
| **与决策联系** | 理论层面的论证结论用于解决现实层面的问题 | 法学的实践品格：论证最终要回应实践 |

### 核心公式

```
问题（现实层面）→ 论证（理论层面）→ 结论 → 决策（现实层面）
```

爱因斯坦："问题无法在其产生的层面被解决。"——需要在理论/原理层面分析问题，然后用结论指导实践决策。法学研究中，不能抱着现实中的现象直接分析，要看到现象背后的理论问题。

## 三、论证的构成要素与术语澄清

### 论证三要素

一个完整的论证包含三个基本要素：

1. **前提（论据/理由）**：为另一个断言提供理由的陈述
2. **结论（论点）**：被前提支持的断言
3. **推理（论证/推论）**：由前提推出结论的过程

### 术语体系对照

论证领域存在多种术语体系，初学者常被困扰。以下是对照：

| 体系 | 支持要素 | 被支持要素 | 连接过程 | 适用场景 |
|------|---------|-----------|---------|---------|
| 三段论 | 大前提+小前提 | 结论 | 演绎推理 | 仅适用于演绎论证 |
| 通用体系 | 前提/论据/理由 | 结论/论点 | 推理/论证/推论 | 各种推理类型 |
| 假设体系 | 前提 | 结论 | 假设（推理成立的基础） | 强调底层信念 |

**注意**："论证"有广义和狭义之分。广义=前提+结论+推理（本章标题）；狭义=推理过程。阅读时需辨别。

## 四、断言——论证的语言基础

论证由**断言（assertion/claim/statement）**构成。断言是指在口头或书面交流中，表明自己观点的、有真假之分的论断。

### 断言 vs 非断言

不是所有句子都是断言。以下类型**不是断言**，不能直接构成论证：

| 非断言类型 | 示例 | 为何不是断言 |
|-----------|------|------------|
| **命令** | "完成作业！" | 意在让对方执行，非判断 |
| **建议** | "我们中午吃火锅好吗？" | 只提供选项，不追求对方认同 |
| **请求** | "希望老师高抬贵手" | 表达希望，非主体判断 |
| **说明** | "苹果是一种常见水果" | 陈述事实，不包含对事物的判断 |

断言必须包含**对事物的判断意图**，最常见形式是"X是Y""X不是Y""X比Y好"。

### 断言的语言要求

1. **词语必须精准**：核心概念在论文中要有可操作定义，避免"在前文用一种含义，到后文发生偷换"
2. **如何精准？——下定义**：尤其是论文的核心概念，哪怕定义有争议，至少在一篇文章内统一用法
3. **区分观点和事实**：观点需要论证，事实需要证据
4. **区分事实和虚构**：学术写作中杜绝虚构

## 五、论证与非论证的五大区分 ⭐

> 这是田洪鋆《批判性思维与写作》最核心的增量知识之一。许多法学论文的论证缺陷，根源在于混淆了论证与以下五种非论证。

### 1. 说明 ≠ 论证

**说明**是陈述事物的状态、特征或过程，不包含"证明"的意图。**论证**是用理由支持结论，目的是让接受者信服。

**法学论文中的表现**：很多学生论文看似在"论证"某个观点，实则只是在"说明"——详细介绍某个制度的内容、某个法条的沿革、某国法的规定，却没有用这些信息来证明任何有争议的命题。

> 📋 诊断："这篇论文把制度介绍完了，但你想证明什么？"

### 2. 价值判断 ≠ 论证

**价值判断**是依据个人价值观作出的评价（"这很好""这不公平"），不提供可检验的理由。**论证**要求理由具有主体间可检验性。

**法学论文中的表现**：论文中反复出现"应当""必须""有必要"，但未给出法学方法论层面的理由（规范依据/体系解释/目的解释/比较法经验等）。

> 📋 诊断："你说法官'应当'考虑比例原则——这个'应当'的规范基础是什么？"

### 3. 修辞 ≠ 论证

**修辞**通过语言感染力引发读者共鸣。**论证**通过逻辑推理使读者信服。修辞可能掩盖论证的跳跃。

**法学论文中的表现**：用比喻、排比、反问等文学手法替代逻辑推理。雷磊指出："写论文不是写诗，要少用修辞，修辞是不精确思维的体现。"

> 📋 诊断："这段读起来很有感染力——但如果去掉修辞，剩下哪些逻辑链条？"

### 4. 说服 ≠ 论证

**说服**是让对方接受某个观点（可以通过情感、权威、利益等任何手段）。**论证**是通过提供可检验的理由和推理使对方信服。

**法学论文中的表现**：诉诸权威（"某大咖说过……"而不分析其推理）、诉诸后果（"如果不接受这个观点就会……"而不证明观点本身的正确性）。

> 关联：`references/legal-argumentation.md` §2（Toulmin模型中的Warrant和Backing）

### 5. 无根据断言 ≠ 论证

**无根据断言**是提出一个观点但不提供任何支持理由。**论证**必须为结论提供前提和推理。

**法学论文中的表现**：论文通篇是"笔者认为……""本文主张……"，但没有法条、判例、学说或数据支撑。

> 📋 诊断："你的核心观点是什么？支撑它的最强论据是什么？如果只能用一条论据说服别人，你选哪条？"

## 六、推理的类型

### 演绎推理（Deduction）

从一般原则推出具体结论。

- **有效论证**：如果前提为真，结论必然为真（逻辑形式正确）
- **可靠论证**：有效论证 + 所有前提实际为真

**法学示例**：
```
大前提：故意杀人者处死刑、无期徒刑或十年以上有期徒刑（刑法第232条）
小前提：被告人故意杀害了被害人
结论：被告人应被判处相应刑罚
```

### 归纳推理（Induction）

从具体事例推出一般结论。归纳论证不追求"必然为真"，而是追求"有力度的"——前提使结论具有较高概率为真。

**法学示例**：
- 案例归纳：分析100份判决 → 归纳出法院在某种情形下的一般裁判倾向
- 比较法归纳：多个法域对同一问题采取类似方案 → 某种做法可能是普遍趋势

### 两者比较

| | 演绎论证 | 归纳论证 |
|---|---------|---------|
| 推理方向 | 一般→具体 | 具体→一般 |
| 结论确定性 | 必然（有效时） | 或然（有力度时） |
| 法学应用 | 法教义学三段论 | 案例实证/比较法 |
| 评价标准 | 有效性+可靠性 | 力度（样本/代表性） |

## 七、批判性阅读：解构论证

批判性阅读是指对文本进行主动的、反思式的阅读，不仅理解"作者说了什么"，更分析"作者的论证是否成立"。

### 阅读的三个层次

1. **信息层**：文本说了什么？（事实、观点、结论）
2. **论证层**：作者如何论证？（前提、推理、结论之间的关系）
3. **批判层**：论证是否成立？（前提可靠吗？推理有效吗？有其他可能性吗？）

### 如何阅读一篇法学论文

1. **先读标题和摘要**：判断论文的核心问题和基本结论
2. **再读引言**：作者如何建构问题？既有研究哪里不足？
3. **读各级标题**：整体论证骨架是什么？
4. **深入正文**：每个分论点→论据→推理链条是否完整？
5. **回到整体**：核心论证是否成立？逻辑上是否有跳跃？

## 八、批判性写作：建构论证

### 从问题到论证

```
问题(现实困惑) → 论题(学术提炼) → 论点(核心主张) → 论证(证明过程) → 结论
```

### "问题→结论→依据"框架（田洪鋆 构思篇）

1. **问题**：明确提出你要解决的问题
2. **结论**：陈述你的核心论点
3. **依据**：为结论提供理由（法条/判例/学说/数据/逻辑推理）

### "提出问题→分析问题→解决问题"框架（论文写作通用结构）

对应法学论文的标准结构：
- **提出问题** = 引言（界定问题+既有研究不足）
- **分析问题** = 正文论证（分论点层层递进）
- **解决问题** = 结论（总结+回应实践）

## 九、批判性思维在法学写作中的应用

### 9.1 "给出原因" vs "推理证明"

> 这是田洪鋆在审查大量硕博论文后总结的最常见问题。

- **给出原因**："我这样认为，因为我觉得……"——主观的、自己相信的理由
- **推理证明**："我的结论基于以下可检验的理由……"——客观的、使别人相信的理由

**法学论文中，每一个主张都需要"推理证明"而非"给出原因"**。你的理由必须来自：
- 法律规范的体系解释
- 判例中的裁判逻辑
- 学理上的公认分析框架
- 可复现的经验数据

### 9.2 自我审查问题清单

在写作和修改时，对每一段论证问：

1. 这段话是论证还是说明？我要证明什么？
2. 我的理由是"自己相信的理由"还是"能使别人相信的理由"？
3. 这段论证中有没有用修辞替代逻辑？
4. 我的核心概念有可操作定义吗？
5. 前提和结论之间有逻辑跳跃吗？（苏二条：话与话之间要经得起推敲）
6. 每句话是否有助于说明论点或为论证提供帮助？（科享二条）

### 9.3 与本skill其他模块的关联

- **选题阶段**：批判性思维帮助你区分question→problem→issue（参见 `topic-selection.md` §1.4）
- **论证阶段**：五大区分直接用于诊断论证质量（参见 `legal-argumentation.md` §4）
- **文献阶段**：批判性阅读用于文献评估和综述（参见 `literature-search.md` §8）
- **行文阶段**：批判性写作框架指导论文结构（参见 `legal-writing.md` §7）
