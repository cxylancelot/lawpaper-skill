# 法学常用表达词库

> 按功能分类的法学学术标准表达。用于行文润色，避免口语化和表达不规范。

---

## 一、提出论点

| 功能 | 标准表达 | 适用场景 |
|------|---------|---------|
| 正面提出观点 | 本文认为 | 提出你的核心论点，最常用 |
| 较弱的提出 | 本文尝试论证 | 你的论点是探索性的或不完全的 |
| 较强的提出 | 本文主张 | 你对一个争议问题采取明确立场 |
| 指出问题 | 有待追问的是 / 尚需回答的是 | 点出现有文献未解决的问题 |
| 指出盲区 | 已有研究普遍忽视了 | 文献批评，引出你的贡献 |

**避免**：
- "笔者认为"：法学论文中"本文"比"笔者"更正式，"笔者"略显个人化
- "我觉得""我认为"：过于口语化
- 在同一篇论文中混合使用"本文""笔者""我们"：选一个并保持一致

---

## 二、引述与对话

| 功能 | 标准表达 | 适用场景 |
|------|---------|---------|
| 正面引述 | XX教授指出 / XX认为 | 引述某学者的观点 |
| 中立引述 | 如XX所言 / 按照XX的分析 | 引述但不评价 |
| 总结立场 | 通说认为 / 主流观点认为 | 概括学界的共识 |
| 引出分歧 | 存在争议的是 / 分歧在于 | 指出有不同观点 |
| 介绍反对观点 | 有学者质疑 / 不同意见认为 | 引出与你不同的观点 |
| 整合多种观点 | 综合来看 / 现有研究大致可分为 | 做文献综述时使用 |

**避免**：
- "有学者认为"（不说是谁）——除非确实找不到具体来源，否则应给出具体姓名
- 在引述后面不加分析直接跳下一句——每一个引述后都应该有"这意味着""这说明了"

---

## 三、限定与让步

| 功能 | 标准表达 | 适用场景 |
|------|---------|---------|
| 范围限定 | 在……范围内 / 就……而言 | 限缩论点的适用范围 |
| 条件限定 | 在……前提下 / 当且仅当 | 说明论点成立的条件 |
| 承认例外 | 当然，这不排除……的可能 | 承认有例外但不推翻主要观点 |
| 承认不确定性 | 此处有待进一步研究 | 诚实标注你未完全解决的问题 |
| 部分同意 | 这一观点有一定的道理，但…… | 先认可对方再提出批评 |
| 尚未涉及 | 这一问题超出了本文的讨论范围 | 明确边界 |

**避免**：
- 没有限定的绝对化表述："毫无疑问""众所周知""必然""显然"——除非真的是无人争议的公理
- "需要说明的是"后面跟的内容并不需要说明——这是被滥用的过渡句

---

## 四、批判与商榷

| 功能 | 标准表达 | 适用场景 |
|------|---------|---------|
| 温和的质疑 | 值得商榷的是 / 这一理解可能忽略了 | 对不同意见的礼貌批评 |
| 较强的批评 | 该观点难以解释 / 无法说明 | 指出对方论证的缺陷 |
| 指出遗漏 | 该分析忽略了 / 未考虑 | 指出对方未处理的重要因素 |
| 指出逻辑问题 | 由此并不能得出…… | 指出推理跳跃 |
| 建设性批评 | 这一分析在……方面是正确的，但在……方面有待修正 | 肯定+批评+改进方向 |
| 辩论式收束 | 综上，A说比B说更具说服力，因为…… | 在比较后给出判断 |

**避免**：
- 空泛的批评："具有局限性"——必须说出具体局限是什么
- 稻草人攻击：批评了一个没人真的主张的弱立场
- 人身化的表述：批评观点，不批评人（"这一观点"而非"该作者"）
- 学术不应使用过于激烈的词汇如"荒谬""可笑的"

---

## 五、总结与过渡

| 功能 | 标准表达 | 适用场景 |
|------|---------|---------|
| 段落总结 | 上述分析表明 / 由此观之 | 总结一段论证后引出结论 |
| 章节过渡 | 在厘清……之后，仍需追问…… | 从一章过渡到下一章 |
| 全文总结 | 综上 / 综上所述 / 归纳全文 | 结论部分的开始 |
| 转向分析 | 那么，接下来要分析的是…… | 开启新的论证环节 |
| 拉回主线 | 回到前述问题 / 回到本文的核心关切 | 分析偏离后拉回 |

**避免**：
- "下面将分析"——生硬且啰嗦，用论证内容的自然过渡替代
- "此外""另外"串联多个段落——说明你的结构是并列的而非递进的

## 六、反模式：口语化 → 学术替代

| 口语化表达 | 学术替代 |
|-----------|---------|
| "说白了" | "简言之""质言之" |
| "大家都知道" | "如所周知""学界已普遍认识到" |
| "笔者觉得" | "本文认为" |
| "很有意思" | "值得关注""具有理论意义" |
| "这样一来" | "由此""以此观之" |
| "其实" | "实则""究其实质" |
| "当然啦" | "当然""不可否认" |
| "可以说" | "可以说""在此意义上"（注意"可以说"容易过度使用） |

## 七、自称规范

法学论文中自称的几种选择及取舍：

| 自称方式 | 风格 | 适用场景 | 注意事项 |
|---------|------|---------|---------|
| **本文** | 中立/客观 | 最通用，法学期刊主流 | "本文认为""本文将论证""本文的分析表明" |
| **笔者** | 略个人化 | 部分学者使用 | "笔者"暗示个人立场，比"本文"更柔和。更常见于台湾法学论文 |
| **我们** | 集体/对话 | 讨论式论文 | 有"邀请读者一起思考"的效果，但在正式论文中需谨慎使用 |
| 避免第1人称 | 最客观 | 极少数期刊偏好 | "据此可以认为""进一步的分析表明"——消除一切人称 |

**核心原则**：选一个并贯彻到底，不要在同一篇论文中换用。

## 八、法学特有术语使用规范

### 引用法条和判决时的表达

| 场景 | 规范表达 | 不规范的表达 |
|------|---------|------------|
| 法条用语 | "《民法典》第X条规定……" | "《民法典》说……" |
| 判决用语 | "法院认定/认为……" | "法官觉得……" |
| 裁判要旨 | "该案的裁判要旨指出……" | "这个案子告诉我们……" |
| 法律适用 | "适用[法条]" | "用[法条]"（过于口语） |
| 不被采纳 | "该主张未被法院采纳" | "法院没理这个" |

### 常用法学术语（表达精确性提醒）

| 术语 | 精确含义 | 常见误用 |
|------|---------|---------|
| 废止 | 法律被正式废除 | 不要与"停止适用"混淆 |
| 停止适用 | 在实际执行中不再适用（但法条形式上可能仍存在） | 不同于正式废止 |
| 修订 | 法律的修改（可能涉及多条） | 不同于"修正"（后者通常是小幅修改） |
| 司法解释 | 最高法/最高检发布的规范性解释 | 不要将部门规范性文件（如会议纪要）称为司法解释 |
| 裁判要旨 | 判例中提炼的法律规则 | 不要将"法院查明"当作"裁判要旨" |
| 举证责任 | 谁承担不能证明的败诉风险 | 不要与"提供证据的责任"混淆 |

### 论证建构用语

| 功能 | 推荐表达 | 避免 |
|------|---------|------|
| 提出主张 | "本文主张……其规范依据在于……" | "笔者认为……"（无后续支撑） |
| 引用并对话 | "XX教授提出……（引文）。这一分析框架在以下方面可以进一步推进：……" | "XX说过……所以这个观点正确" |
| 限定/让步 | "在……情形下，上述结论需要受到以下限缩：……" | "当然也有例外"（不具体） |
| 批评/商榷 | "XX的观点在……方面提供了重要启示，但在……问题上，其论证存在以下不足：……" | "XX的观点是错误的"（笼统否定） |
| 过渡/推进 | "上述分析解决了……问题，但随之而来的一个更深层的问题是：……" | "下面将分析……""接下来将介绍……" |

### 区分说明与分析

| 说明（应避免） | 分析（应使用） |
|-------------|-------------|
| "该制度包括三个要件：A、B、C。" | "该制度的三个要件中，A要件的认定在实践中争议最大，原因在于……" |
| "德国法规定……日本法规定……" | "德日两国对同一问题采取不同方案，其制度选择的逻辑差异在于……" |
| "《民法典》第X条规定……" | "《民法典》第X条的规范结构显示，立法者在……问题上采取了……立场" |

### 批判性评价用语

| 强度 | 表达 |
|------|------|
| 弱（提出疑问） | "值得追问的是……""这里可能忽略了一个前提……" |
| 中（指出问题） | "这一论证在……环节存在逻辑跳跃""该结论的前提假设是……，但这一假设未必成立" |
| 强（根本性质疑） | "如果……的前提不成立，该理论的整体框架就需要重新审视" |
